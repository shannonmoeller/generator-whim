# `whim`

[![NPM version][npm-img]][npm-url] [![Downloads][downloads-img]][npm-url] [![Tip][amazon-img]][amazon-url]

A protean boilerplate generator based on my mercurial preferences.

## Install

    $ npm install --global whim

## Usage

### Module

    $ whim module

### Website

    $ whim website
